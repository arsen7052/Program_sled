# 🛡️ Program_sled

> Система мониторинга и контроля доступа для образовательных учреждений

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## 📋 Описание

**Program_sled** — это Python-приложение для автоматизированного контроля использования цифровых ресурсов в учебных заведениях. Система использует распознавание лиц для идентификации учащихся, блокирует запрещённые приложения и веб-сайты, а также ведёт журнал нарушений в реальном времени.

### 🔑 Ключевые возможности

| Функция | Описание |
|---------|----------|
| 👤 Распознавание лиц | Идентификация учащихся по лицу с помощью dlib и OpenCV |
| 🚫 Блокировка приложений | Автоматическое завершение процессов запрещённых программ |
| 🌐 Блокировка сайтов | Закрытие вкладок браузера с нежелательным контентом |
| 📅 Расписание | Работа системы в соответствии с учебным расписанием групп |
| 📡 Интеграция с бэкендом | Отправка данных о нарушениях на сервер через REST API |
| 🎥 Работа с веб-камерой | Захват изображения для верификации пользователя |
| ⚙️ Многопоточность | Фоновое обновление списков блокировок без остановки мониторинга |
| 📦 Сборка в EXE | Поддержка компиляции в исполняемый файл через PyInstaller |
